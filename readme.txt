项目简介：
本项目基于STM32F103微控制器和FreeRTOS实时操作系统开发，在传统双轮自平衡小车系统上进行了优化，为智慧校园提供了物流解决方案。本系统通过 MPU6050 获取角度信息，结合 PID 算法控制电机，实现自平衡控制。同时支持蓝牙通信控制、OLED 显示、超声波测距等功能，采用多任务调度、DMA+中断优化通信效率，具备较强的实时性与稳定性。
项目特色与优化：
1.FreeRTOS多任务管理：自定义控制任务、蓝牙任务、OLED显示任务、超声波测距任务等，并支持任务通知机制。
2.动态内存优化：自定义heap_4.c中的动态内存管理算法，实现空闲内存块延迟合并机制。
3.任务安全机制：启用任务栈溢出检测vApplicationStackOverflowHook()。
4.蓝牙通信优化：使用DMA+空闲中断(UART IDLE)实现高效串口接收。接收缓冲区通过全局变量与任务共享，避免频繁中断处理。数据接收完毕后通过任务通知唤醒蓝牙处理任务。
